# Cyclistic-bike-share-analysis-case-study
Analysis of 5.4 million bike-share trips using R

## Overview    
Analysis of 5.585.259, after the cleaning 5.438.287 bike-share trips from Cycling company in Chicago to understan how casual riders  and annual members use bikes differently and design a marketing strategy to convert casual riders into members.  

## Business question  
How do casual riders and annual members use Cyclistic bikes differently?  

## Tools used  
**R** - data cleaning, analysis and visualization;  
**tidyverse** packages - data wrangling;  
**lubridate** packages - data manipulation;  
**janitor** packages - data cleaning;  
**ggplot2** - data visualization;  
**Google slides** - presentation.  

## Data
 * Cycling trip data - public source;  
 * Jan 2025 - Dec 2025 analyzed period;  
 * 5.585.259 trips collected, 5.438.287 trips after the cleaning, removed almost 2.7% data.  

## Key Findings
 * casual riders take 63% longer trips than members;  
 * casual riders peak on weekends and members on weekdays;  
 * the number of trips from casual riders drops considerably in winter.  

## Recommendations
 * a "weekend membership" for the casual riders;  
 * marketing campain in **Spring** and **Summer**;  
 * emphasize casual riders how much they would save with a membership.
